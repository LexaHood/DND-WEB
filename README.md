Для работы с монорепозиторием:
1. Установить yarn
```sh
  npm i -g yarn
```
2. В корне проекта установить зависимости для фронтенда и бекенда
```sh
  yarn install
```

**Запуск проекта**

 - Для запуска фронтенда используется команда
```sh
  yarn start:frontend
```
- Для запуска бекенда используется команда
```sh
  yarn start:backend
```
